# Two-Qubit Transmon Design using Qiskit Metal

## Overview

This repository contains a personal learning project focused on the design of superconducting transmon qubits using Qiskit Metal. The goal of this project is to gain practical experience with quantum hardware design concepts by creating and visualizing a simple two-qubit transmon architecture.

This project was completed as part of my self-learning journey in quantum computing and superconducting circuit design. It is intended for educational purposes and serves as a foundation for future work involving resonators, coupling mechanisms, electromagnetic simulations, and larger quantum processor architectures.

## Project Structure

```text
.
```

### Files

* **two_qubit_design.py**

  * Contains the Qiskit Metal code used to generate the two-transmon qubit layout.

* **report.txt**

  * Brief project report describing the design, objectives, and key concepts explored during implementation.

* **images/**

  * Contains screenshots and visualizations of the generated qubit layout.
